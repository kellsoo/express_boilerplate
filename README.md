# Express boilerplate

MVC bolierplate for express with EJS template system

# Project structure

```.env
# Environment variables
PORT = 3000
IP = 127.0.0.1
```

## install dependencies

```bash
npm install
```

## Run

```bash
npm start
```

## Run dev

```bash
npm run dev
```
